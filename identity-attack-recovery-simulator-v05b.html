Screen 02 of 03 — Device Code Flow
The victim is redirected to a page showing a device code and the real Microsoft URL login.microsoft.com/device. This is a legitimate Microsoft OAuth device code flow — abused. The spinner says "Waiting for you to sign in." The victim enters the code on the real Microsoft page.
⚠ No malware. No credential harvesting. This is a legitimate Microsoft authentication flow being weaponized.
